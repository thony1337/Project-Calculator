# JavaScript Calculator

A basic calculator built with HTML, CSS, and JavaScript. 
Supports addition, subtraction, multiplication, division, 
and chained operations.

- Users click number buttons to enter digits.
- The first number is stored in `numberA`.
- When an operator is clicked, the calculator:
  1. Stores the operator.
  2. If a second number exists (`numberB`), it calculates using the previous operator.
  3. Displays the result and prepares for the next operation.
  4.
